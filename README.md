# React + Vite + TypeScript

Ett modernt projekt med React, Vite och TypeScript.

## Kom igång

### Installera beroenden

```bash
npm install
```

### Starta utvecklingsserver

```bash
npm run dev
```

Öppna [http://localhost:5173](http://localhost:5173) i din webbläsare.

### Bygg för produktion

```bash
npm run build
```

### Förhandsgranska produktionsbygg

```bash
npm run preview
```

## Teknologier

- **React** - UI-bibliotek
- **Vite** - Byggverktyg och utvecklingsserver
- **TypeScript** - Typad JavaScript
